My Blog
================

My [blog](http://ashfurrow.com).

Setup
----------------

```shell
git clone https://github.com/ashfurrow/ashfurrow.github.io.gitgit
cd blog
bundle install
rake
```

Then navigate to [http://0.0.0.0:4567](http://0.0.0.0:4567).
