My Blog
================

My [blog](http://ashfurrow.com).

Setup
----------------

```shell
git clone https://github.com/AshFurrow/blog.git
cd blog
bundle install
rake
```

Then navigate to [http://0.0.0.0:4567](http://0.0.0.0:4567).
